# booQ-UI
![CI](https://github.com/traPtitech/booQ-UI/workflows/CI/badge.svg)

booQ Frontend

## 開発環境構築
実行にはNode.jsが必要です

```shell
$ npm i
$ npm run gen-api
```

## コマンド
### ホットリロード環境
```shell
$ npm run dev
```

**実行前に一度`$ npm run gen-api`を実行する必要があります**

### ビルド
```shell
$ npm run build
```
ビルド

```shell
$ npm run gen-api
```
APIクライアントコード生成
